--> Alert to users that are using an out of date version of Firefox and more.
--> Alerta a los usuarios que están utilizando una versión desactualizada de Firefox y más.

== Description ==

When a user use a not updated version of Firefox, the plugin add on footer page a meseagge alerting to him for update her Mozilla Firefox browser. Too show a brief description of problem and the links for download the last stable version and why. 

After you install Actualiza Firefox, you must go to control panel "Actualiza Firefox" under "Settings" and configure it.

-=-=- -=-=- -=-=- -=-=- -=-=- -=-=- -=-=- -=-=- -=-=- -=-=-

Cuando un usuario utiliza una versión no actualizada de Firefox, el plugin añade en el pie de página un mensaje de alerta para que él actualice su navegador Mozilla Firefox. También muestra una breve descripción del problema y brinda los enlaces para descargar la última versión estable y por qué hacerlo.

Después de instalar Actualiza Firefox, debes ir al panel de control "Actualiza Firefox" bajo el menú configuración" y configurararlo.

== Installation ==

1. Upload the folder "actualiza-firefox" to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Update settings.

== Update ==

1. Deactivate the plugin.
2. Delete all old files and folders.
3. Upload the folder "actualiza-firefox" to the `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Update settings.

== Instalación ==

1. Sube la carpeta "actualiza-firefox" al directorio `/wp-content/plugins/`
2. Activa el plugin a través del menú 'Plugins' de WordPress.

== Actualización ==

1. Desactiva el plugin.
2. Borrar todos los archivos y carpetas antiguos.
3. Sube la carpeta "actualiza-firefox" al directorio `/wp-content/plugins/`
4. Activa el plugin a través del menú 'Plugins' de WordPress.
5. Actualiza la configuración.
